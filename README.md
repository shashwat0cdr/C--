## 🛠️ Getting Started

### Prerequisites
- C++ compiler (g++, clang, or MSVC)
- Text editor or IDE (VS Code, Visual Studio, Code::Blocks, etc.)

### Compilation
```bash
# Navigate to the desired directory
cd Intro/day_01

# Compile the C++ file
g++ -o program_name filename.cpp

# Run the compiled program
./program_name
```
